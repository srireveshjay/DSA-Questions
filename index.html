DSA Question Bank

A curated, growing collection of Data Structures & Algorithms problems — organized by topic, tagged by difficulty, and updated regularly.

How to add questions

  1. Open index.html in your editor.
  2. Scroll to the DATA block near the bottom of the <script> tag.
  3. Find the section you want (e.g., arrays) and add a new object: { num: "...", name: "...", difficulty: "easy|medium|hard", platform: "LC|GFG|CF|..." }
  4. Save and push to GitHub — GitHub Pages will update automatically.
