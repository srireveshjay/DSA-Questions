How to add a question

  1. Open index.html and find the DATA object in the <script> tag.
  2. Inside the target section's questions array, add a new object with: num, name, difficulty, platform, problem, examples, constraints, approach, code fields.
  3. Save and push to GitHub — the page updates instantly.
